🔐 Image Encryption Tool using Pixel Manipulation
This project is a basic image encryption tool developed using Python. It demonstrates fundamental cybersecurity concepts by encrypting and decrypting images through pixel manipulation techniques.

🚀 Features
Encrypt images using pixel swapping (permutation)
Apply mathematical operations (pixel value shifting)
Decrypt images using the same key
Maintains original image after successful decryption

🧠 How It Works
The tool uses two main techniques:
Permutation: Swaps pixel positions to distort the image structure
Substitution: Modifies pixel values using a key (shift operation)
During decryption, the process is reversed to restore the original image.

💻 Technologies Used
Python
Pillow (PIL) library

📌 Usage
Run the program
Choose encrypt or decrypt
Provide input image path
Enter output file name
Enter key value (0–255)
View the encrypted/decrypted image

🔐 Learning Outcome
This project helps understand basic image encryption techniques, key-based security, and the concepts of permutation and substitution used in cybersecurity.
